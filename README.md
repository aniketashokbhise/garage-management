# 🚗 Workshop Manager

A modern and responsive **Garage / Workshop Management System** built using the **MERN Stack**. This application helps garage owners efficiently manage customers, vehicles, employees, inventory, invoices, attendance, and workshop operations from a single dashboard.

---

## 🌐 Live Demo

🚀 **Project Link:**  
**https://garage-management-m3y7t6neb-aniketashokbhises-projects.vercel.app/**

## ✨ Features

### 📊 Dashboard
- Revenue Overview
- Monthly Revenue
- Pending Jobs
- Low Stock Alerts
- Revenue Trend Chart
- Payment Status Summary

### 👥 Customer Management
- Add, Edit & Delete Customers
- Search Customers
- Customer Vehicle History

### 🚗 Vehicle Management
- Register Vehicles
- Service History
- Vehicle Details
- Customer-Vehicle Mapping

### 👨‍🔧 Employee Management
- Employee Registration
- Employee Profile
- Monthly Salary
- Salary Advance
- Employee Loan
- Remaining Balance
- Bonus
- Overtime
- Payment Status

### 📅 Attendance Management
- Date-wise Attendance
- Mark Present, Absent & Leave
- Attendance History
- Monthly Attendance Report

### 📦 Inventory Management
- Add Spare Parts
- Stock In / Stock Out

### 🧾 Invoice Management

### ⚙️ Settings

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/workshop-manager.git
```

### Install Frontend

```bash
cd frontend
npm install
```

### Install Backend

```bash
cd backend
npm install
```

---

## 🔐 Environment Variables

### Backend (.env)

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
CLIENT_URL=http://localhost:5173
```

### Frontend (.env)

```env
VITE_API_URL=http://localhost:5000/api
```
